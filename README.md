Arrow keys  - move in direction  
l  - look  
i  - inspect  
q  - quit  
t  - torch on/off  
s  - swim  
f  - fire arrow  
m  - mine  
c  - cut tree/vine  
C  - cast spell  
r  - reveal shallow water  
H  - highlight terrain  
w  - set warp location  
p  - phasewalk  
R  - rest spirit  
d  - detect number of monsters  
M  - spawn a monster nearby  
F1 - background colour on/off  
F2 - hide/show controls
